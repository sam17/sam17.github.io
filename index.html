Moving Baseline Stereo for Fisheye cameras

Autonomous Robotics and Human-Machine Systems Lab
Max Planck Institute of Biological Cybernetics, Germany

The project aims to develop a multi-robot decentralised 3D tracking system using aerial robots. I developed a stereo from fish-eye camera using plane-sweep stereo and am currently working on shifting this into a variable moving baseline using robot poses. All processing is being done on-board Odroid processor which runs Telekyb which is an open-source framework for aerial robot control.
I also developed a ROS Wrapper for ARM Boards for Basler Ace USB 3.0 Camera and designed Camera Holder Parts which were 3D printed.

Mentor: Dr. Aamir Ahmad, Post-Doctoral Fellow, Max Planck Institute of Biological Cybernetics

Report: Moving Baseline Stereo for Fish-eye camera system. (Report in Progress)

Fish-Eye Camera based Indoor Localisation

Aerial Robotics Kharagpur
Centre for Excellence in Robotics, IIT Kharagpur

International Aerial Robotics Competition(IARC) Mission VII aims to extend the state of the art of aerial robotics in GPS Denied environments. It requires robots to localise themself without ground features or GPS indoors. I am attempting to use fish-eye camera feed due to it's high field of view to get the edges of the room as landkarks and use the curvature of those edges to estimate self position of the robot. Inspired from the bijection proven in 3D to 2D bijection for spherical objects under equidistant fisheye projection, Aamir Ahmad et.al.

Guide : Prof. Jayanta Mukhopadhyay, Computer Science and Engineering Department, IIT Kharagpur
Mentor: Dr. Aamir Ahmad, Post-Doctoral Fellow, Max Planck Institute of Biological Cybernetics

Report: Fisheye Image based localisation in GPS denied environments. (Report in Progress)

    Multi-Agent Decentralised Sensor Converage

    Swarm IIT Kharagpur
    Centre for Excellence in Robotics, IIT Kharagpur

    The project aims to develop a novel swarm system of robots where tasks and robot types can be plugged in as modules and the system handles the execution and mutual localisation. The first task the project aims at is maximising sensor coverage in known map. So, I am working on an algorithm based on algebraic topology to cover maximum sensor coverage in a known map using decentralised set of robots. The idea is to optimise coverage problem and expand to patrolling in the map to reduce the uncovered time by a robot in an area.

    Currently, I am trying to run this on our simulator which will then be expanded to real robots.

    Guide : Prof. Pallab Dasgupta, Computer Science and Engineering Department, IIT Kharagpur

    Image Analysis to Study Draining in Mini-Channels

    Bachelor Thesis Project
    Department of Chemical Engineering

    Liquid draining is a very common phenomenon with applications in every industry. However, draining in mini-channels is a relatively new area of research with no experimental data to validate models developed for draining in micro-channels. This project aimed to develop a model for draining in partially open mini-channels used extensively in bio-medical applications.

    I developed a model which was then validated using experimental data obtained by tracking the liquid-air interface as liquid flows using image analysis techniques on video feed. The video feed records draining in mini-channel tubes at multiple inclinations and different number of needles attached to partially open the tube.

    Guide : Prof. S.Ray and Prof. Gargi Das, Department of Chemical Engineering, IIT Kharagpur

    Mosaiq - Automated Image Mosaicing of Torn Paper Documents

    Inter-Hall Open Software, IIT Kharagpur

    Mosaiq is a image mosaicing software which can join torn paper documents together. It is also capable of giving partial reconstruction if complete was unsuccessful. Human input could also be used when reconstruction is not possible.

    I implemented a globally consistent reconstruction method using a turning function. A set of candidate matches was found by comparing Turning Functions of different torn pieces. From these matches, global consistency is achieved by gradient projection. This was one of the methods implemented to test the performance of the software.

    Text to Braille Converter

    Inter-Hall Hardware Exhibition

    Text to Braille Converter was a device which took A4 sheet printed pages over which a CMOS sensor would scroll over to get the image. The image was then converted to a character whose braille was then printed on a platform driven by 30 servo motors. All processing was performed on 8-bit ATMEGA microcontrollers.

    I designed the OCR module which used naive but fast method so that it can work on 8-bit microcontrollers, testing it initially using openCV and then shifting to AVR. Data transmission of images was also optimised by converting to Bayer for faster communication.
